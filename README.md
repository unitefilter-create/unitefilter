# UNITE FILTER AI Filtration Selector

An AI-assisted industrial filtration system selection tool for process engineers.

## 🚀 What it does

- Input process conditions
- Automatically recommend filtration systems
- Compare top 3 solutions
- Help engineers reduce design time

## ⚙️ Features

- Rule-based decision engine
- Industrial equipment database
- Scoring recommendation system
- Lightweight frontend (no backend required)

## 🧠 Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

## 📦 How to run

Just open index.html in browser.

## 🌍 Use Case

- Chemical industry
- Oil & Gas
- Pharmaceutical
- Water treatment
